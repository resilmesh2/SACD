This application demos how to use the Sentinel Layout library in combination with lazy loaded modules as individual agendas.

The layout is configurable, please consult the source code and README to see details.

The layout is also responsive. You can try it displaying this page on a mobile device or in dev tools.

You can generate new notifications by clicking on the button with plus sign inside a bell icon. You can display previously generated notifications through the regular bell icon. All notifications are stored in the IndexedDB of your browser.
